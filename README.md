# Un Oui Pour La Vie — Site vitrine

Site vitrine complet d'un **Wedding Planner** fictif, réalisé en **HTML / CSS pur** (sans framework) dans le cadre d'une SAÉ de 1ʳᵉ année de BUT Informatique (IUT de Lens).

Projet d'équipe — DeMassary · Chirez · Maroille — conçu après analyse de sites de wedding planners de référence, avec un accent sur l'intégration web, le design responsive et la structuration sémantique.

## Pages

- **Accueil** (`Homepage/`) — présentation et mises en avant
- **À Propos** (`APropos/`) — histoire de l'agence + vidéo de présentation
- **Services** (`Services/`) — lieu, traiteur, photographe, artiste, EVG, lune de miel, forfaits…
- **Boutique** (`Boutique/`) — décoration, papeterie, maquillage, accessoires, véhicules, vêtements…
- **Blog** (`Blog/`)
- **Contact** (`Contact/`)
- **Connexion** (`Connexion/`) et **Compte** (`Compte/`)

## Stack

HTML5 sémantique, CSS3 (mise en page responsive), aucune dépendance ni JavaScript.

## Lancer le site

Ouvrir `Homepage/HomePage_UnOuiPourLaVie.html` dans un navigateur.
